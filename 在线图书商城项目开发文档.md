# 在线图书商城项目开发文档

# 一、项目概述

本项目为在线图书商城系统，采用前后端分离架构，后端基于SpringBoot \+ Controller \+ Service \+ Mapper \+ Pojo分层实现，前端基于Vue\.js构建，实现了用户端购书全流程与管理员端后台管理功能，包含用户认证、图书管理、购物车、订单交易、数据可视化等核心模块。本项目由一人独立开发完成，覆盖前后端全栈开发所有内容。

# 二、开发人员与职责说明

本项目由单人独立开发完成，全面负责项目所有模块的设计、编码、测试与交付，具体职责如下：

- 后端核心层：SpringBoot项目搭建、用户认证、权限拦截、数据库连接池与工具类封装

- 后端图书层：图书查询、详情展示、搜索联想、CRUD与文件上传（基于SpringBoot实现接口）

- 后端交易层：购物车、订单管理、收藏功能与支付逻辑（基于SpringBoot接口开发）

- 前端用户端：前台Vue页面、Axios请求封装、表单校验与交互

- 前端管理端：后台Vue页面、ECharts数据可视化、权限判断

# 三、技术栈

## 3\.1 后端技术

- 核心框架：SpringBoot（替代Java Servlet，简化配置，实现接口开发）

- 分层架构：Controller \+ Service \+ Mapper \+ Pojo（基于SpringBoot规范搭建）
  - Controller：接收请求，返回响应
  - Service：处理业务逻辑
  - Mapper：数据库持久化操作
  - Pojo：JavaBean实体类，对应数据库表结构

- 数据库：MySQL

- 连接池：JDBC连接池（集成于SpringBoot项目）

- 工具类：Token生成与校验、SpringBoot拦截器（替代LoginFilter实现权限拦截）

- 其他：文件上传（SpringBoot自带MultipartFile实现）、统一编码处理、全局异常处理

## 3\.2 前端技术

- 框架：Vue\.js

- 网络请求：Axios

- 数据可视化：ECharts

- 构建工具：Vue CLI / Vite

- 样式：CSS / Less

# 五、核心技术实现细节

## 5\.1 权限控制

- 通过SpringBoot拦截器（LoginInterceptor）拦截未登录请求，重定向至登录页，配置拦截路径与放行路径

- 基于TokenUtil生成用户令牌，存储于请求头或Cookie中，拦截器中校验令牌有效性

- 区分普通用户与管理员角色，管理员端路由需额外校验角色权限，通过SpringBoot接口返回角色信息，前端进行权限控制

## 5\.2 数据库连接

- 在SpringBoot项目中集成JDBC连接池，通过配置文件（application\.yml/properties）配置连接参数，提升性能与稳定性

- 封装JDBCUtil工具类，实现连接获取与释放的统一管理，集成于SpringBoot容器，供Mapper层调用

## 5\.3 搜索联想

- 后端基于MySQL模糊查询实现关键词匹配，通过SpringBoot Controller暴露接口，返回联想结果

- 前端通过防抖处理减少请求频次，调用SpringBoot后端接口，实时展示联想结果

## 5\.4 数据可视化

- 管理员端集成ECharts，展示图书销量、订单金额等统计图表

- 后端基于SpringBoot开发聚合查询接口，封装图表所需数据格式，供前端ECharts调用渲染

# 六、数据库表结构设计

## 6\.1 数据库总览

本项目使用MySQL数据库，命名为bookstore，包含以下核心表：

- user：用户信息表

- book：图书信息表

- cart：购物车表

- orders：订单主表

- order_item：订单明细表

- favorite：收藏表

- category：图书分类表

## 6\.2 表结构详情

### 6\.2\.1 用户表（user）

存储用户/管理员的账号、密码、角色等信息。

| 字段名      | 类型           | 约束                                                           | 说明                         |
| ----------- | -------------- | -------------------------------------------------------------- | ---------------------------- |
| user_id     | INT            | PRIMARY KEY AUTO_INCREMENT                                     | 用户ID（主键）               |
| username    | VARCHAR\(50\)  | NOT NULL UNIQUE                                                | 用户名（唯一）               |
| password    | VARCHAR\(100\) | NOT NULL                                                       | 密码（加密存储）             |
| phone       | VARCHAR\(20\)  | NOT NULL UNIQUE                                                | 手机号（唯一）               |
| role        | TINYINT        | NOT NULL DEFAULT 1                                             | 角色：1\-普通用户，2\-管理员 |
| create_time | DATETIME       | NOT NULL DEFAULT CURRENT_TIMESTAMP                             | 创建时间                     |
| update_time | DATETIME       | NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间                     |

### 6\.2\.2 图书分类表（category）

存储图书的分类信息，用于分类筛选。

| 字段名        | 类型          | 约束                               | 说明                      |
| ------------- | ------------- | ---------------------------------- | ------------------------- |
| category_id   | INT           | PRIMARY KEY AUTO_INCREMENT         | 分类ID（主键）            |
| category_name | VARCHAR\(50\) | NOT NULL                           | 分类名称                  |
| parent_id     | INT           | DEFAULT 0                          | 父分类ID（0表示一级分类） |
| create_time   | DATETIME      | NOT NULL DEFAULT CURRENT_TIMESTAMP | 创建时间                  |

### 6\.2\.3 图书表（book）

存储图书的详细信息，包括封面、价格、库存等。

| 字段名      | 类型            | 约束                                                           | 说明                       |
| ----------- | --------------- | -------------------------------------------------------------- | -------------------------- |
| book_id     | INT             | PRIMARY KEY AUTO_INCREMENT                                     | 图书ID（主键）             |
| book_name   | VARCHAR\(100\)  | NOT NULL                                                       | 图书名称                   |
| author      | VARCHAR\(50\)   | NOT NULL                                                       | 作者                       |
| category_id | INT             | NOT NULL FOREIGN KEY REFERENCES category\(category_id\)        | 所属分类ID                 |
| price       | DECIMAL\(10,2\) | NOT NULL                                                       | 图书价格                   |
| stock       | INT             | NOT NULL DEFAULT 0                                             | 库存数量                   |
| cover_img   | VARCHAR\(255\)  |                                                                | 封面图片路径               |
| description | TEXT            |                                                                | 图书描述                   |
| status      | TINYINT         | NOT NULL DEFAULT 1                                             | 上架状态：1\-上架，0\-下架 |
| create_time | DATETIME        | NOT NULL DEFAULT CURRENT_TIMESTAMP                             | 创建时间                   |
| update_time | DATETIME        | NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间                   |

### 6\.2\.4 购物车表（cart）

存储用户购物车中的商品信息。

| 字段名                                       | 类型     | 约束                                                           | 说明                           |
| -------------------------------------------- | -------- | -------------------------------------------------------------- | ------------------------------ |
| cart_id                                      | INT      | PRIMARY KEY AUTO_INCREMENT                                     | 购物车项ID（主键）             |
| user_id                                      | INT      | NOT NULL FOREIGN KEY REFERENCES user\(user_id\)                | 所属用户ID                     |
| book_id                                      | INT      | NOT NULL FOREIGN KEY REFERENCES book\(book_id\)                | 图书ID                         |
| quantity                                     | INT      | NOT NULL DEFAULT 1                                             | 购买数量                       |
| create_time                                  | DATETIME | NOT NULL DEFAULT CURRENT_TIMESTAMP                             | 创建时间                       |
| update_time                                  | DATETIME | NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间                       |
| UNIQUE KEY uk_user_book \(user_id, book_id\) |          |                                                                | 同一用户同一图书仅能有一条记录 |

### 6\.2\.5 订单主表（orders）

存储订单的核心信息，关联用户与订单状态。

| 字段名           | 类型            | 约束                                            | 说明                                                            |
| ---------------- | --------------- | ----------------------------------------------- | --------------------------------------------------------------- |
| order_id         | VARCHAR\(32\)   | PRIMARY KEY                                     | 订单号（主键，使用UUID生成）                                    |
| user_id          | INT             | NOT NULL FOREIGN KEY REFERENCES user\(user_id\) | 下单用户ID                                                      |
| total_price      | DECIMAL\(10,2\) | NOT NULL                                        | 订单总金额                                                      |
| receiver_name    | VARCHAR\(50\)   | NOT NULL                                        | 收货人姓名                                                      |
| receiver_phone   | VARCHAR\(20\)   | NOT NULL                                        | 收货人电话                                                      |
| receiver_address | VARCHAR\(255\)  | NOT NULL                                        | 收货地址                                                        |
| order_status     | TINYINT         | NOT NULL DEFAULT 1                              | 订单状态：1-待发货，2-待收货，3-已收货，4-已取消 |
| pay_time         | DATETIME        |                                                 | 支付时间                                                        |
| deliver_time     | DATETIME        |                                                 |                                                                 |
